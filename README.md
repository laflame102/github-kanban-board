# github-kanban-board

GitHub repo issues viewer as a kanban board

## How to use

1. **Clone the repository**

   ```bash
   git clone https://github.com/laflame102/github-kanban-board
   cd github-kanban-board
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start**

   Run the development port

```bash
 npm run dev
```

4. **Open app link**

```bash
http://localhost:5173
```
